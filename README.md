# Punith S Portfolio

A premium AI-powered HR operations portfolio built with Next.js, Tailwind CSS, Framer Motion, GSAP-ready structure, and Three.js visualization.

## Features
- Cinematic hero experience with live KPI cards
- HR analytics command center and QA system thinking panels
- GitHub repo integration and analytics summaries
- Dynamic project showcases with enterprise SaaS styling
- Advanced dark fintech UI with glassmorphism and motion

## Install

```bash
cd /Users/punith.s/Projects/punith-portfolio
npm install
npm run dev
```
